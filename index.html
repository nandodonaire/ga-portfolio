About Me

Hi! My name is Fernando Donaire. I'm a full-stack web developer and recently completed a full-time Immersive Web Development program at General Assembly.

Prior to this I was working at a software startup doing technical support, but wanted to pursue a more creative career. Working closely with developers at my previous job (many of whom also didn't have a formal CS background) really inspired me learn to code. In my own free time I started learning a little about coding, found that I really enjoyed it, and decided to enroll in the web development course at GA.

I decided to pursue a career in web development because I was intrigued by the idea of creating my own web applications rather than troubleshooting existing ones. I love coding because it's the perfect combination of creativity and problem solving. To me, there are few things more exciting than working hard on a project and seeing it come together.

BitStashclose

This was my first team project at General Assembly which I worked on as part of a 4 developer team. We built a full-stack data storage application (similar to Dropbox) which allows users to store files on an Amazon S3 bucket. Frontend was built using JavaScript, JQuery, Bootstrap, Handlebars, and Ajax. Backend API was built using Express, Node.js, Mongoose, and AWS.

This was probably my favorite project that I worked on during my time at GA. While working on this project I learned a great deal about how to work with a team of developers, how to ensure that we are all on the same page, how to keep our focus on our end goal, and how to pair-program.

BrewJournalclose

A full-stack web application that allows users to keep track visits to breweries, the beers sampled at each brewery, as well notes about their visit. I built the frontend client using Ember.js and the API was built using Ruby on Rails.

This project was challenging because it was my first time using Ember.js which we had learned the week before starting the project. I decided to challenge myself to use Ember.js and ended up really enjoying working with it.

This was my first full-stack application. I built an app for keeping track of TV shows and movies that users are interested in seeing in the future as well as where they can see them.
